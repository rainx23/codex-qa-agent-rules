# 业务 Profiles

面向特定业务或测试场景的补充规则。

## 目录定位

在核心规则之下为 API、禅道、权限、金融和 Web UI 等场景提供专项约束。

## 主要内容

| 路径 | 作用 | 是否手工维护 |
| --- | --- | --- |
| `*.md` | 场景规则和专项门禁 | 是 |

## 使用入口

- 由 `AGENTS.md` 或对应 Skill 根据任务场景加载。

## 维护约束

- 新增或调整 Profile 时说明与核心规则的关系，并补充受影响测试、根 README 和 CHANGELOG。
- 本目录不含自动生成文件。

版本与完整变更历史统一见 [CHANGELOG.md](../../CHANGELOG.md)。
