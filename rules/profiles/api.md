# API Profile

只对证据命中的接口和调用链生成测试点。

## 契约

- 核对方法、路径、版本、请求头、认证方式、参数名称、类型、必填、默认值和枚举。
- 核对响应状态码、业务码、结构、字段类型、空值、分页、错误模型和兼容策略。
- 新增字段检查旧客户端忽略能力；删除、重命名和类型变化检查上下游兼容。
- 不用当前实现替代接口说明，不虚构请求或响应。

## 行为

- 覆盖合法、缺失、非法、边界、重复、乱序和超大请求。
- 明确幂等键、重复提交、重试、超时、取消、限流、降级、熔断和补偿。
- 写操作检查事务、部分成功、回滚、最终一致性和重复消费。
- 查询接口检查过滤、排序、分页、权限和数据范围。

## 调用链

- 公共接口变更搜索直接调用方、封装层、网关、前端、任务、消息和外部调用方。
- 检查接口自动化、契约测试、Mock 和文档是否同步。
- 无法确认调用方时记录证据缺口和关联回归，不宣称无影响。
