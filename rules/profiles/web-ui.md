# Web UI Profile

只扫描需求、Diff、代码或历史缺陷命中的能力，不把本文件逐项转换为 TC。

## 查询

- 明确查询入口、默认条件、生效时机、条件组合、清空和重置。
- 验证请求只携带已生效条件，返回集合只包含符合条件的数据。
- 验证空值、特殊字符、边界长度、日期边界、多选交并关系和无结果。
- 翻页、排序、刷新、返回页面后检查条件和结果状态是否按已确认规则保持。
- 快捷查询与普通查询并存时检查覆盖、合并和隔离规则；未说明时进入待确认点。

## 列表和表格

- 验证列名、数据来源、格式、单位、精度、空值、枚举、排序和分页。
- 检查表头、列表行、汇总条、展开行、弹窗和导出的同一字段口径。
- 检查跨页选择、全选范围、固定列、横向滚动、展开收起和大数据量。
- 不把已有列表可打开、按钮可点击作为逻辑变更的主用例。

## 字段和个性化配置

- 扫描字段显示隐藏、全选反选、拖动排序、输入序号移动、固定列和恢复默认。
- 检查保存、另存为、应用、取消、重置、删除、重命名、搜索和切换模板。
- 检查刷新、重新进入、退出登录、切换账号后的持久化。
- 检查个人、角色、组织和公共模板的隔离及冲突优先级。
- 新增、删除或重命名字段时检查历史模板兼容，不默认推断导出跟随页面配置。
- 多字段规则相同则合并为等价类用例；保存方式、权限或持久化结果不同才拆分。

## 弹窗

- 验证触发行与弹窗对象一致，不出现跨行、上次缓存或并发串数据。
- 检查打开、关闭、取消、确认、重复打开、遮罩、滚动和尺寸边界。
- 多个弹窗共享数据源、条件和预期时使用公共入口合并；数据源或异常处理不同才拆分。

## 跳转和下钻

- 验证来源行、路由参数、目标范围、返回路径和权限。
- 检查空参数、失效对象、越权对象、重复点击和新旧页面兼容。
- 目标页筛选和汇总必须与来源对象保持已确认口径。

## 导出

- 明确导出范围是当前页、筛选结果、选中项还是全部数据。
- 验证字段、顺序、格式、精度、时区、权限、脱敏和页面口径一致。
- 检查空结果、大数据量、重复提交、超时、失败提示、文件名和编码。
- 未说明导出口径时只记录待确认点，不自行默认。

## 状态、配置和交互

- 用决策表或状态迁移覆盖合法、非法、重复和回退路径。
- 配置开关检查默认值、生效范围、生效时机、灰度、缓存和回滚。
- 检查加载、禁用、错误提示、焦点、键盘、浏览器缩放和可访问性。
- 文案和纯视觉变化仅在影响验收或交互时生成用例。
